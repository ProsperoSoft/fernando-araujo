# Fernando Araujo - Tarjeta de Contacto Digital

[![GitHub Pages](https://img.shields.io/badge/website-up-brightgreen)](https://prosperosoft.github.io/fernando-araujo)

Landing page minimalista para mi tarjeta de presentación física.

## ✨ Características

- 📇 Guardado automático de contacto (vCard)
- 💬 Enlace directo a WhatsApp
- ✉️ Email con un clic
- 🌐 Portafolio web
- 📱 100% responsive
- ⚡ Ultraligero y rápido

## 🚀 Ver la página

## 🛠️ Tecnologías

- HTML5 / CSS3
- JavaScript vanilla
- GitHub Pages
- vCard 3.0
